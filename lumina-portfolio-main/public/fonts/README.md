# Hero font: Caslon Graphique

Place **Caslon Graphique** (Wealthsimple-style) here for the title and intro:

- **Filename:** `CaslonGraphique.woff2`
- **Path:** `public/fonts/CaslonGraphique.woff2`

The site will use it for the hero heading and tagline. If the file is missing, the fallback is Georgia / serif.

Caslon Graphique is a commercial font (Font Bureau). You need a license to use it on the web. After licensing, export or convert your font to WOFF2 and save it with the filename above.
